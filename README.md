# Factory Effects Pack

Небольшой мод, который добавляет визуальные эффекты для ванильных производственных зданий.

Эффекты взяты из мода [Mindustry Overhaul](https://github.com/KOCTOLOM/Mindustry-overhaul) by KOSTOLOM и переупакованы в отдельный мод с `hidden: true`, что делает его совместимым с серверами — он не добавляет новый контент, только визуал.

## Затронутые здания

Silicon Smelter, Graphite Press, Kiln, Melter, Pulverizer, Spore Press, Pyratite Mixer, Blast Mixer, Coal Centrifuge, Phase Weaver, Surge Smelter, Plastanium Compressor, Multi-press, Silicon Crucible, Cryofluid Mixer

## Установка

Скиньте `.zip` в папку `mods/` и перезапустите игру.

- **Windows:** `%appdata%/Mindustry/mods/`
- **Linux:** `~/.local/share/Mindustry/mods/`
- **Android:** `Mindustry/mods/`
